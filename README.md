# 🏥 Healthcare Analytics Dashboard

## Overview
The Healthcare Analytics Dashboard is an interactive Tableau project designed to analyze patient demographics, medical conditions, healthcare trends, and risk indicators. The dashboard transforms raw healthcare data into actionable insights through intuitive visualizations and KPI-driven reporting.

## Objectives
- Analyze patient demographics and healthcare patterns.
- Monitor medical condition distribution across patients.
- Evaluate cholesterol-related risk factors.
- Perform gender-based healthcare analysis.
- Support data-driven healthcare decision-making.

## Tools & Technologies
- Tableau Public
- Microsoft Excel
- Data Visualization
- Dashboard Development
- Healthcare Data Analytics

## Dashboard Highlights

### Healthcare Overview Dashboard
- Total Patients Analysis
- Average Age & Cholesterol KPIs
- Patient Distribution by Condition
- Gender Distribution
- Patient Visit Trends

### Gender-Based Healthcare Analysis
- Male, Female, and Other Patient Metrics
- Medical Conditions by Gender
- Demographic Analysis by Age Group
- Gender Distribution Insights

### Healthcare Risk & Demographic Insights
- Cholesterol Risk Distribution
- Average Cholesterol by Age Group
- Medical Condition Analysis
- Risk-Oriented Healthcare Insights

## Key Insights
- Healthcare conditions were distributed across multiple patient groups, with certain conditions showing higher prevalence.
- Patient demographics remained relatively balanced across gender categories.
- Cholesterol levels varied among age groups, highlighting potential risk patterns.
- Interactive dashboards improved the interpretation of healthcare data and trends.
